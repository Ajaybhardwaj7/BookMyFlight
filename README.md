
# Flight Booking System
This is a sample app to book flights.
 
 NPM v3.10.8
 
 Node v6.11.2
 
 Chrome v60.0.3112.101

# Server
1. cd server
2. Install dependencies. Run > npm i
3. Run > npm start
4. Server is running at port 3002 (localhost)

# Client
1. cd client
2. Install dependencies. Run > npm i
3. Run > npm start
4. Launch http://localhost:9001
